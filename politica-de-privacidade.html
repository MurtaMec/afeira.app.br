Política de Privacidade da Plataforma Feira

Versão 1.0 — 19 de fevereiro de 2026

Em conformidade com a LGPD (Lei 13.709/2018) e o Marco Civil da Internet (Lei 12.965/2014).

1. Identificação do Controlador

A plataforma FEIRA é a controladora dos dados pessoais tratados conforme esta Política.

Contato do Encarregado (DPO): feiraappsuporte@gmail.com

2. Escopo

Esta Política de Privacidade aplica-se a todos os serviços oferecidos pela Plataforma, incluindo:

  • Aplicativo mobile (Android e iOS)
  • Páginas web associadas
  • Sistemas internos de gestão

4. Dados Coletados

4.1. Dados fornecidos diretamente pelo Usuário

  • Nome completo e razão social
  • Endereço de e-mail
  • Senha (armazenada com hash bcrypt)
  • CPF/CNPJ
  • Telefone
  • Endereço e localização
  • Chave PIX
  • Perfil profissional (tipo de atividade, produtos, certificações)
  • Imagens (foto de perfil, produtos, documentos)
  • Mensagens enviadas via chat

4.2. Dados coletados automaticamente

  • Geolocalização (com consentimento)
  • Informações do dispositivo (modelo, sistema operacional, versão do app)
  • Dados de uso do aplicativo (páginas acessadas, funcionalidades utilizadas)
  • Histórico de transações e pedidos
  • Interações com outros usuários (avaliações, indicações)

4.3. Dados sensíveis

Dados biométricos, quando utilizados para autenticação, são processados localmente no dispositivo do Usuário e NUNCA são coletados ou armazenados pela Plataforma.

4.4. Menores de idade

A Plataforma não é destinada a menores de 18 anos. Não coletamos intencionalmente dados de menores.

5. Finalidades do Tratamento

Base Legal Finalidades
Execução contratual Criação de conta, processamento de pedidos, comunicação entre usuários, faturamento, geração de relatórios
Interesse legítimo Prevenção a fraudes, melhoria dos serviços, análises estatísticas, segurança da plataforma
Obrigação legal Cumprimento de obrigações fiscais, cooperação com autoridades, retenção de registros conforme Marco Civil da Internet

6. Compartilhamento de Dados

6.1. Com outros Usuários

Dados de perfil público (nome, tipo, localização, produtos, avaliações) são compartilhados com outros usuários da Plataforma para viabilizar as transações comerciais.

6.2. Com Operadores (prestadores de serviço)

  • Supabase — infraestrutura e banco de dados
  • Expo — notificações push e distribuição do app
  • Sentry — monitoramento de erros
  • Google / Apple — autenticação social

6.3. Com autoridades

Mediante ordem judicial ou requisição legal fundamentada.

NUNCA vendemos dados pessoais. O compartilhamento é restrito ao estritamente necessário para a prestação dos serviços.

7. Segurança dos Dados

Adotamos medidas técnicas e organizacionais para proteger os dados pessoais:

  • Criptografia AES-256 para dados sensíveis em repouso
  • HTTPS/TLS para dados em trânsito
  • Bcrypt para hash de senhas
  • Row Level Security (RLS) no banco de dados
  • SECURITY DEFINER para funções com acesso controlado
  • Auditoria e logs de acesso
  • Sentry para monitoramento de incidentes

9. Direitos do Titular

Nos termos da LGPD, o Titular dos dados tem direito a:

  • Confirmação da existência de tratamento
  • Acesso aos dados pessoais
  • Correção de dados incompletos, inexatos ou desatualizados
  • Anonimização, bloqueio ou eliminação de dados desnecessários ou excessivos
  • Portabilidade dos dados a outro fornecedor
  • Eliminação dos dados tratados com consentimento
  • Informação sobre compartilhamento com terceiros
  • Revogação do consentimento
  • Revisão de decisões automatizadas
  • Oposição ao tratamento em caso de descumprimento da LGPD

Prazo de resposta: até 15 (quinze) dias úteis.

Canal de atendimento: feiraappsuporte@gmail.com

11. Incidentes de Segurança

Em caso de incidente de segurança que possa acarretar risco ou dano relevante aos Titulares, a Plataforma comunicará:

  • A Autoridade Nacional de Proteção de Dados (ANPD) em até 72 horas
  • Os Titulares afetados, com descrição da natureza dos dados, riscos e medidas adotadas

12. Decisões Automatizadas

A Plataforma utiliza processos automatizados para:

  • Cálculo de reputação — baseado em avaliações e histórico
  • Atribuição de selos — verificação de identidade e qualidade
  • Cálculo de taxa — percentual sobre transações
  • Otimização de rotas — cálculo de distâncias e logística
  • Detecção de atividade suspeita — prevenção a fraudes

O Titular pode solicitar a revisão de qualquer decisão automatizada por uma pessoa natural, no prazo de 15 (quinze) dias.

13. Alterações nesta Política

A Plataforma poderá atualizar esta Política de Privacidade a qualquer momento. Alterações substanciais serão comunicadas com pelo menos 30 (trinta) dias de antecedência por meio dos canais oficiais.
